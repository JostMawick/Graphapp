Compiling Programs

On X-Windows:

  • Use the supplied examples Makefiles to help you.
  • There are Makefiles for Unix and some Windows compilers.

Windows 98, 2000, ME and NT

  • The library must be compiled first.
  • Then, each program must be linked to the library.
  • GraphApp is not a DLL, don't put it in your Windows directory.

Surfing the Web

  • GraphApp can be downloaded via the web
  • Location is http://enchantia.com/software/graphapp/
  • The example programs are in doc/examples/

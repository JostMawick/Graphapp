Tools
-----

This directory contains some utility programs or tools
which may help a programmer.

A list of the tools, together with a short description,
follows. For more information, read the source code
for each tool. Each tool is in a single source code file
so each should be compiled separately.

addres.c    Adds a set of resources (files) to an app.
getres.c    Extracts one named resource from an app.
seeres.c    Displays a list of all of the resources in an app.

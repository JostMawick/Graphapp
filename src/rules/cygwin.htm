How to make Graphapp with CygWin

Copy the appropriate makefile to graphapp\src
Use cygwin32.mak if using CygWin for Windows 32-bit.
Use cygwinx.mak if using CygWin for X-Windows.
Rename the file to makefile
Go to graphapp\src

make static
This will create the static library.

make dynamic
This will create the DLL.

make demo
Go to demo\ (type cd demo) and type imagine *.png

make demo2
This creates another demo program. Go to demo\ and type tester

make demo3
Go to demo\ and type viewutf8 example.utf8

make demo4
Go to demo\ and type imgtest

make demo5
Go to demo\ and type blend

Simple Examples

  • alldemo.c - A demonstration of many of the features of the package. [Screen Shot]
  • char.c - A program to display Unicode 'sub-fonts'. [Screen Shot]
  • clock.c - A simple, tiny digital clock. [Screen Shot]
  • fields.c - Places three text fields on a window. [Screen Shot]
  • listtest.c - Demonstrates display of text inside a list box. [Screen Shot]
  • pizza.c - A program for ordering pizza, using some controls. [Screen Shot]
  • scribble.c - Demonstrates drawing using colours, and handling mouse events. [Screen Shot]
  • scrolls.c - Places many scroll bars in a window. [Screen Shot]
  • tabpane.c - A program for displaying tabbed panels. [Screen Shot]
  • viewutf8.c - Demonstrates loading and drawing UTF-8 encoded text. [Screen Shot]

Drawing Examples

  • arcs.c - Draws arcs of various sizes and line thicknesses. [Screen Shot]
  • blend.c - Demonstrating alpha-blending and drawing to images. [Screen Shot]
  • ellipses.c - Draws ellipses of various sizes and line thicknesses. [Screen Shot]
  • fastline.c - Draws lines of various lengths and thicknesses, and times how long it takes, to compare drawing speeds. [Screen Shot]
  • imgtest.c - Draws various images (some of which are partially transparent) across the window, to demonstrate image copying. [Screen Shot]
  • moire.c - Shows Moire patterns using lines and the timer. [Screen Shot]
  • polygons.c - Draws polygons which tile to form smooth shapes. [Screen Shot]
  • rainbow.c - A colour-chooser window. [Screen Shot]
  • smiley.c - Draws a smiley face using various drawing operations. [Screen Shot]
  • spectrum.c - Draws a spectrum of colours across a window. [Screen Shot]

Complex Examples

  • editdraw.c - Demonstrates drawing using XOR mode, and using the mouse to draw figures using "rubber banding". [Screen Shot]
  • imagedit.c - An image editor program which allows you to edit single pixels, or modify an image's colour palette. [Screen Shot]
  • imagine.c - A complex program which allows loading and display of images in various image formats. [Screen Shot]
  • tester.c - A large but simple program which demonstrates creating multiple window and controls, drawing, and handling user actions. [Screen Shot]
  • utf8edit.c - A simple UTF-8 text editor. [Screen Shot]
